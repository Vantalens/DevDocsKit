# 需求到发布追踪矩阵模板

版本：v1.0.0  
状态：模板  
最后更新：2026-04-19

## 使用说明
用于建立端到端可追踪关系：需求 -> 流程 -> 任务 -> 实现 -> 测试 -> 发布。

## 矩阵字段
- REQ_ID
- FLOW_ID
- TASK_ID
- 实现对象（组件/API/脚本）
- TEST_ID
- 发布版本
- 状态
- 备注

## 矩阵样例
| REQ_ID | FLOW_ID | TASK_ID | 实现对象 | TEST_ID | 发布版本 | 状态 | 备注 |
|---|---|---|---|---|---|---|---|
| REQ-001 | FLOW-001 | TASK-001 | login API | TEST-001 | v0.1.0 | done | 首版登录 |
| REQ-001 | FLOW-002 | TASK-002 | login page | TEST-002 | v0.1.0 | done | 含异常态 |

## 校验规则
- 每个 REQ_ID 至少映射 1 个 FLOW_ID
- 每个 FLOW_ID 至少映射 1 个 TEST_ID
- 状态字段只允许：todo/in_progress/done/blocked
