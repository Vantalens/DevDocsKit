# 门禁阈值分级建议（个人版）

版本：v1.0.0  
状态：生效  
最后更新：2026-04-19

## 1. 目标
按个人能力成长分阶段提高质量阈值，保证可执行与可持续。

## 2. 分级
- S1 起步期
- S2 稳定期
- S3 强化期

## 3. 指标建议
| 指标 | S1 | S2 | S3 |
|---|---|---|---|
| 单元覆盖率 | >= 40% | >= 55% | >= 70% |
| 核心流程通过率 | 100% | 100% | 100% |
| P1 通过率 | >= 90% | >= 95% | >= 98% |
| 接口 P95 | <= 1000ms | <= 800ms | <= 600ms |
| 发布前检查 | 100% | 100% | 100% |

## 4. 升级条件
- 连续 2 次迭代达标可升级一级

## 5. 降级条件
- 连续 2 次发布出现阻断或回滚失败
