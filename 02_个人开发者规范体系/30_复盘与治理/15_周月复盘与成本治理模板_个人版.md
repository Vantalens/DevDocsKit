# 周月复盘与成本治理模板（个人版）

版本：v1.0.0  
状态：模板  
最后更新：2026-04-19

## 1. 目标
建立稳定的个人改进节奏，控制技术债与成本风险。

## 2. 周复盘
- 本周完成事项
- 未完成事项与原因
- 质量问题与根因
- lessons 新增条目
- 下周前三优先任务

## 3. 月复盘
- 目标达成率
- 发布成功率
- 回滚次数与原因
- 性能与稳定性趋势
- 依赖与架构债清单

## 4. 成本治理
每月记录：
- 云服务成本
- 数据库存储与流量
- 第三方 API 成本
- AI Token 成本

优化动作：
- 成本异常项
- 下月降本目标
- 执行动作与负责人（本人）

## 5. 评分建议
- 交付效率（30）
- 质量稳定（30）
- 可维护性（20）
- 成本控制（20）
