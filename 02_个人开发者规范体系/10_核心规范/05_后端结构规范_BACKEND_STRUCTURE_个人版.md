# 后端结构规范 BACKEND_STRUCTURE（个人版）

版本：v1.0.0  
状态：生效  
最后更新：2026-04-19

## 1. 目标
确保数据结构、接口契约和安全策略可维护且可追踪。

## 2. 数据模型
每个实体必须记录：
- 实体名
- 字段与类型
- 主键与索引
- 关系
- 生命周期字段

## 3. 接口契约
每个接口必须记录：
- API ID
- URL 与 Method
- 请求参数校验
- 响应结构
- 错误码
- 鉴权方式

## 4. 事务与一致性
- 关键写操作必须明确事务边界
- 并发冲突必须有处理策略

## 5. 安全基线
- 服务端输入校验
- 最小权限
- 敏感字段脱敏
- 日志可追踪

## 6. 评审清单
- 是否定义完整错误码
- 是否有迁移回滚方案
- 是否覆盖权限边界

## 7. 变更记录
- v1.0.0：首版。
